Bus booking project

A simple way to find buses and manage routes.

BusBuddy is a Django web app for route search, bus registration, bookings, and a simulated payment flow. It is built as a practical college project with passenger and conductor workflows.

What you can show in the demo

The project is intentionally simple, but it covers a complete backend workflow.

Passenger search

Search by source, destination, date, operating day, AC type, seating type, and driver information.

Bus registration

Conductors can add route details, stops, timings, total seats, and weekly or date-based schedules.

Booking workflow

The app creates bookings, calculates available seats, simulates payment, and stores booking history.

How it works

Built around two real users

A passenger searches and books a seat. A conductor creates a route and keeps track of schedules. This makes the project easy to explain in a viva because every model has a clear purpose.

  • Login and registration are handled by Django authentication.
  • Routes, stops, schedules, bookings, and payments are stored as models.
  • Templates render the pages directly from backend data.
