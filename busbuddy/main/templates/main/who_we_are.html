About the project

A practical Django bus booking system

BusBuddy was built as a college submission project to demonstrate authentication, database models, forms, route search, booking, and dashboard workflows.

Project objective

To create a working bus booking platform where passengers can search routes and operators can register buses with schedules.

Technology focus

The project uses Django, SQLite, Django templates, static CSS/JavaScript, authentication, and server-rendered forms.
